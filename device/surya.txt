Flashing steps:
Clean:
- Wipe data, dalvik and cache
- Flash Blaze
- Format data
- Reboot system

Dirty:
- Wipe dalvik and cache
- Flash Blaze
- Reboot system

Notes:
- It's already Gapps build so flashing gapps package isn't necessary
- Be sure to use Recovery that supports A14
- GAPPS and Firmware included 
- Not all kernel support UQPR2. Check kernel compatibility before flashing
